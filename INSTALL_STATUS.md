# 🚀 Harbor & Voyage 프로젝트 성공적으로 설치 완료!

## ✅ 설치 및 설정 완료 사항

### 📦 패키지 설치
- Next.js 14.2.31 (최신 보안 패치 적용)
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion (애니메이션)
- Lucide React (아이콘)
- Swiper (슬라이더)

### 🔧 설정 수정
- PowerShell 실행 정책 설정
- Next.js 보안 취약점 수정
- TypeScript 설정 최적화
- Tailwind CSS 커스텀 컬러 설정

### 🎨 구현된 기능
- ✅ 라이트/다크 테마 전환
- ✅ 반응형 헤더 네비게이션
- ✅ Hero 섹션 (낮/밤 전환 애니메이션)
- ✅ About 섹션 (브랜드 스토리)
- ✅ Menu 섹션 (낮/밤 메뉴 탭)
- ✅ Events 섹션 (이벤트 및 예약 폼)
- ✅ Gallery 섹션 (호버 효과)
- ✅ Contact 섹션 (위치 정보)
- ✅ Footer
